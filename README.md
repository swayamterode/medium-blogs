## Updating the Commons NPM Package

To update the Commons NPM package, follow these steps:

-Update the **version** in the `package.json` file.

- Build the package using TypeScript compiler:

```bash
   tsc -b
```

- Publish the package to NPM:

```bash
   npm publish
```
